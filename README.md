# sft
LLMs and Supervised Fine-Tuning (SFT), explaining how pretrained models—initially trained on broad text corpora—can be adapted for specific NLP tasks. While pretrained models demonstrate general capabilities across diverse tasks, SFT refines them to become more useful, action-oriented, and domain-specialized for targeted applications.
